# Pokelist

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.2.7.

## Install dependences
Run the following for install dependences necessary.

```shell
    npm install
```

## Run the application
Run the following for start application 

```shell
    ng serve --open
```
